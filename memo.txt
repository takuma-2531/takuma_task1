let sum = numbers.reduce(0) { num1, num2 -> Int in
    num1 + num2
}
が
let sum = numbers.reduce(0, +)
と書き換えられる。
->分かった

+記号は
static func + (lhs: Int, rhs: Int) -> Int
でIntに定義されている関数。
reduceの第二引数の関数の型？と一致している。
だから+を使うことができる。

では、+と言う関数とは
普通に 1 + 2 とかにすると3が返ってくる。
でも
Int.+(1, 2)
とかにしても、エラーとなり3は返ってこない。
何故？？
